# sample-data-graph

A sample Apollo GraphQL server with a [Users service](https://github.com/gurunate/users-service) REST data source.

## Setup

### Install

```bash
$ npm i
```

### Run

```bash
$ npm start
```

#### Playground

-   http://localhost:4000
